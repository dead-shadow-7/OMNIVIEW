🛣️ Extract Big Roads (Sentinel-2)

Use this feature to extract major road networks from Sentinel-2 satellite imagery. Upload a GeoTIFF (Sentinel-2 or similar) and the AI model will generate:

  • Original image (PNG preview)
  • Processed binary road mask (PNG)
  • Overlay image with detected roads highlighted

Notes: Large TIFFs can be sizable — processing may take several minutes depending on image resolution and your system. If you run into memory or timeout issues, try a smaller tile or run the model locally with the instructions in the README.
